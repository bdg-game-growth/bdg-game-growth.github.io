Registration Problems: Detailed Troubleshooting (Stepwise)

OTP delay or not received

  1. Wait 60–90 seconds before retry.
  2. Check SMS inbox & blocked numbers folder.
  3. Switch between mobile data and Wi-Fi.
  4. Restart phone if nothing works.
  5. Contact support with transaction reference (if visible).

Number already registered

Agar number pehle se registered hai to use login flow. Kabhi-kabhi users multiple accounts create karne ki koshish karte hain — avoid this for compliance and bonus eligibility.

Registration stuck on loading

  • Clear browser cache / use incognito
  • Try mobile browser instead of in-app webviews
  • Ensure official APK is latest version (if using app)

If you repeatedly fail verification, support may ask for KYC/document verification. Follow official steps only — never share sensitive info on social channels.
